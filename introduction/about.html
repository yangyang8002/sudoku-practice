关于本项目

南京师范大学附属中学 2024 级学生研究性学习作品

作品名称：数独练习 —— 基于 PHP + JavaScript 的在线数独平台。

作品简介

  • 支持唯一解与非唯一解两种生成模式，满足不同训练需求。
  • 内置计时排行榜，可将最佳成绩永久保存并查看前 20 名。
  • 提供候选数标记功能，方便高阶玩家排除法解题。
  • 自适应深色 / 浅色系统主题，手机端体验流畅。
  • 纯单文件部署，零依赖，可轻松嵌入任何 PHP 环境。

技术栈

后端：PHP 8.x（仅用于生成题目与排行榜存储）
前端：原生 ES6 + CSS Grid + Flexbox

作者

南京师范大学附属中学 2024 级益智游戏中的数学原理 研究性学习小组
